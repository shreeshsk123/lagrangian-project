# Lagrangian Method for One Constraint

## 📌 Description
This project demonstrates solving optimization problems using the Lagrangian method with one constraint.

## ⚙️ Features
- Solves constrained optimization problems
- Shows step-by-step calculations
- Easy to understand implementation

## 🚀 How to Run
1. Download or clone the repository
2. Open the project in VS Code
3. Run the main file

## 📂 Files Included
- Code files
- PPT presentation
- Poster
- Documentation

## 📊 Conclusion
This project helps in understanding how Lagrange multipliers are used in real-world optimization problems.
